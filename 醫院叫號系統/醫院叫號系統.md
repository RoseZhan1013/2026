# 智慧醫療門診叫號系統 - 開發規格文件

本文件包含智慧門診叫號系統的完整前端網頁原始碼（單一 HTML 檔案架構）。本版面專為醫療診間、醫學中心電子看板與護理師控制台複合載具設計，採用現代化醫療科技感（Clinical Tech）視覺風格，並具備即時數據驅動與智慧語音播報功能。

## ⚙️ 系統核心功能亮點
1. **精簡型高對比看板**：縮小原有的叫號大區塊，轉為橫向精緻橫幅（3.8rem 字級），在不犧牲辨識度的情況下為臨床操作留出更多視覺空間。
2. **語音廣播整合**：串接瀏覽器內建 Web Speech API，一鍵呼叫時自動發出沉穩、清晰的繁體中文語音（例：「心臟內科第一診，請 2 號到診間看診」）。
3. **物件導向排隊模型**：病患清單採用結構化資料（內含診號 `num`、姓名 `name`、掛號類別 `type`），便於未來對接 HIS (醫院資訊系統) 數據鏈。
4. **過號插隊機制**：提供過號一鍵登錄，並支援病患臨櫃報到時由護理師快速將其安排回等候佇列的第一順位（準備看診）。
