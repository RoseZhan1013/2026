# 🐱 喵不可言！貓咪翻翻樂遊戲網頁開發企劃與程式碼

這是一份完整的貓咪主題翻翻牌遊戲開發檔案。本網頁採用原生 JavaScript (ES6+)、HTML5 與 CSS3 開發，完全不依賴任何外部框架，並具備優雅流暢的 3D 翻牌動畫與莫蘭迪紫視覺風格。

---

## 📌 開發需求規格

1. **卡牌規格**：由 16 張卡牌組成，共 8 對完全相同的可愛貓咪高清圖片。
2. **洗牌機制**：內建 Fisher-Yates Shuffle 隨機洗牌演算法，確保每局遊戲體驗皆不重複。
3. **數據追蹤**：即時統計並渲染玩家的「翻牌次數」與「成功配對組數」。
4. **鎖定機制**：防止非法連續點擊。當兩張牌不配對時，會鎖定面板 1 秒鐘，防止暴力破解。
5. **視覺風格**：以「莫蘭迪紫」為主色調，兼具精美質感與俏皮元素，支援跨裝置響應式網格（Responsive Grid）。

## ⚡ 3. JavaScript 邏輯 (`script.js`)

```javascript
// 1. 定義 8 種不同的貓咪精選圖片網址（來源：Unsplash 高清貓咪圖庫）
const catImages = [
    'https://unsplash.com',
    'https://unsplash.com',
    'https://unsplash.com',
    'https://unsplash.com',
    'https://unsplash.com',
    'https://unsplash.com',
    'https://unsplash.com',
    'https://unsplash.com'
];
let cardsArray = [...catImages, ...catImages];

// 2. 遊戲狀態追蹤變數
let flippedCards = [];
let flipCount = 0;
let matchCount = 0;
let isBoardLocked = false; // 鎖定機制：防止非法連續點擊

// 3. 獲取 DOM 元素
const gameBoard = document.getElementById('game-board');
const flipCountDisplay = document.getElementById('flip-count');
const matchCountDisplay = document.getElementById('match-count');
const resetBtn = document.getElementById('reset-btn');

// 4. 洗牌演算法 (Fisher-Yates Shuffle)
function shuffle(array) {
    for (let i = array.length - 1; i > 0; i--) {
        const j = Math.floor(Math.random() * (i + 1));
        [array[i], array[j]] = [array[j], array[i]];
    }
    return array;
}

// 5. 初始化與生成遊戲面板
function initGame() {
    gameBoard.innerHTML = '';
    flippedCards = [];
    flipCount = 0;
    matchCount = 0;
    isBoardLocked = false;
    
    flipCountDisplay.textContent = flipCount;
    matchCountDisplay.textContent = `${matchCount} / 8`;

    // 洗牌
    const shuffledCards = shuffle([...cardsArray]);

    // 動態產生 16 張內含 <img> 的卡牌結構
    shuffledCards.forEach((imgUrl, index) => {
        const card = document.createElement('div');
        card.classList.add('card');
        card.dataset.img = imgUrl; // 儲存網址做為比對標記
        card.dataset.index = index;

        card.innerHTML = `
            <div class="card-face card-back"></div>
            <div class="card-face card-front">
                <img src="${imgUrl}" alt="Cute Cat" class="cat-img">
            </div>
        `;

        card.addEventListener('click', flipCard);
        gameBoard.appendChild(card);
    });
}

// 6. 翻牌核心邏輯
function flipCard() {
    if (isBoardLocked) return;
    if (this.classList.contains('matched')) return;
    if (flippedCards.length === 1 && flippedCards.dataset.index === this.dataset.index) return;

    this.classList.add('flipped');
    flippedCards.push(this);

    if (flippedCards.length === 2) {
        flipCount++;
        flipCountDisplay.textContent = flipCount;
        checkMatch();
    }
}

// 7. 配對檢查與鎖定機制控制
function checkMatch() {
    const [card1, card2] = flippedCards;

    // 比對兩張牌的圖片網址
    if (card1.dataset.img === card2.dataset.img) {
        card1.classList.add('matched');
        card2.classList.add('matched');
        
        matchCount++;
        matchCountDisplay.textContent = `${matchCount} / 8`;
        
        flippedCards = [];

        if (matchCount === 8) {
            setTimeout(() => alert(`🎉 太強了！您總共翻動了 ${flipCount} 次成功通關！`), 500);
        }
    } else {
        isBoardLocked = true;

        setTimeout(() => {
            card1.classList.remove('flipped');
            card2.classList.remove('flipped');
            
            flippedCards = [];
            isBoardLocked = false;
        }, 1000); // 留 1 秒讓玩家記憶圖片
    }
}

resetBtn.addEventListener('click', initGame);
document.addEventListener('DOMContentLoaded', initGame);
```

## 💡 技術重點摘要
* **防破解鎖定**：運用 `isBoardLocked` 狀態。在配對失敗時，面板強制鎖定 1 秒鐘，期間所有點擊無效，完全避免暴力破解行為。
* **滿版防變形圖片**：CSS 使用 `object-fit: cover` 確保任何外鏈貓咪圖片皆可完美填滿正方形卡牌而不壓縮扭曲。
* **防作弊設計**：在圖片樣式加入 `-webkit-user-drag: none` 與 `user-select: none`，防止玩家用滑鼠拖曳卡牌背面圖片、或全選文字時提前洩漏答案。
